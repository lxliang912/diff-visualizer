# Diff Visualizer

可视化查看 Git 分支间的文件变动和提交历史。

## 功能

- **分支对比**: 选择两个分支，快速查看所有文件变动
- **文件树视图**: 以目录结构展示变动文件，支持展开/收起
- **提交历史**: 图形化展示分支间的提交记录
- **快速 Diff**: 点击文件直接打开 VSCode 内置 Diff 视图

## 使用方法

1. 点击侧边栏的 **Diff Visualizer** 图标
2. 选择基准分支和目标分支
3. 查看文件变动列表和提交历史
4. 点击文件查看详细差异
